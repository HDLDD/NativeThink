# NativeThink 开发日志

## 2026-07-11

### 文章阅读 📖
- **4 种内容源**：书籍(公版英文原著) / 刊物(AI生成双语) / 演讲(24篇完整全文含TED) / Wikipedia搜索
- **目录选择**：点击书籍→先显示章节目录→选择章节开始阅读
- **双语对照**：中英逐段对齐显示，段落间紫色翻译
- **按句朗读**：逐句播放 + 预加载下一句，流畅不卡顿
- **等级转换**：AI 将文章改写为初/中/高级
- **Wikipedia 搜索**：英文条目 API 搜索 → 结果列表 → 可添加到"我的列表"保存
- **阅读历史**：自动记录，可回看/清空
- **阅读器**：点击查词 + 翻页(滚轮/按钮) + 字号调节 + 全屏沉浸

### 词汇深度 📚
- **学习 & 复习统一 6 种模式**：闪卡 / 选择题 / 拼写 / 听写 / 配对 / 填空
- **学习 tab**：每日学习量无上限自定义 + 模式一键切换
- **复习 tab**：SM-2 间隔记忆 + 紫色主题独立界面
- **词库浏览**：双列布局 + 筛选芯片 + 列表位置记忆 + 自动朗读
- **搭配学习**：双列布局 + 短语记忆 + 翻译 + 列表位置记忆
- **语块库**：双列布局 + 内置/AI生成 + 记忆筛选 + 自动朗读 + 列表位置记忆
- **切换词书**：页面顶部醒目的词书切换卡片，随时更换
- **位置记忆**：词库浏览/搭配学习/语块库均支持列表滚动位置+选中项记忆，刷新保留

### 登录 & 云同步 🔄
- 注册/登录功能 + Cloudflare KV 存储
- 电脑 ↔ 手机数据自动同步（学习进度、收藏、统计数据等）
- Cloudflare Pages 部署支持

### 性能优化 🚀
- **网页加载**: 全部页面 React.lazy 懒加载，主包 **2.3MB → 897KB (-61%)**
- **词库引擎**: 索引缓存 + 同步 localStorage 检查 + 后台预加载
- **搜索优化**: useDeferredValue 延迟搜索，输入不阻塞 UI

### 朗读引擎 🔊
- **多引擎架构**: SpeechSynthesis → CF Edge TTS → Google TTS
- 按句朗读 + 预加载下一句（无缝衔接）
- 超时兜底机制（onEnd 不可靠时自动推进）
- 手机端 TTS 修复

### 移动端适配 📱
- 底部 Tab 导航栏 + 响应式布局 + 移动端间距优化

### UI 改进 ✨

- 词库浏览筛选芯片 + 词数汇总
- Sticky 置顶 + 分页管理 + 自动朗读
