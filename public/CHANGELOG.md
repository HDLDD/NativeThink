# NativeThink 开发日志

## 2026-07-11 (晚间更新)

### 朗读引擎 (TTS) 🔊 — 重大重构
- **三层引擎架构**: SpeechSynthesis (桌面主引擎) → Cloudflare Function 代理 (手机) → Edge WebSocket → Google TTS
- 修复 **cancel 竞态条件**：`speak()` 中 `ssCancel()` 和 `speakSS()` 双重 cancel 导致切换单词后静默
- `setTimeout(0)` 延迟一个微任务确保 cancel 完全处理后再 speak
- 添加 **1.5s 启动超时检测**：小米/华为浏览器 SpeechSynthesis API 存在但不发声时自动切换到 Cloudflare 代理
- **Cloudflare Pages Function** (`/api/tts`)：Cloudflare 全球节点代理 Google TTS，绕过 GFW，CDN 永久缓存

### 搭配短语查询 ⚡
- **即时反馈**：点击搭配短语立即弹出对话框显示短语，释义异步加载
- **后台预加载**：选中单词后自动静默预加载所有搭配短语到缓存，后续点击秒开

### UI 调整 ✨
- **闪卡缩小**：宽度 448→384px，高度 340→260px，字号和间距相应缩小，手机端更友好
- **学习/复习界面视觉差异化**：学习模式 emerald 色系，复习模式 violet 色系

## 2026-07-11 (日间)

### 性能优化 🚀
- **网页加载速度**: 全部页面改为 `React.lazy` + Suspense 懒加载，主包 **2.3MB → 897KB (-61%)**
- **Dashboard 优化**: `MOCK_CHUNKS` (146KB) 改为动态 `import()`，首页不再捆绑整个语块库
- **词库引擎重构**: 移除冗余 `cleanWordEntry` 映射，预建 `_levelIndex` 索引，`getAllWords()` 惰性缓存
- **同步缓存检查**: `loadLevel()` 同步检查 localStorage 命中则立即返回，二次访问秒开
- **搜索优化**: `useDeferredValue` 延迟搜索，输入不阻塞 UI 渲染
- **后台预加载**: 引导向导挂载时立即在后台加载全部词库，消除"开始学习"后的等待

### 词书 & 学习流程 📚
- 所有 tab 统一使用 wizard 选定的词书，移除独立等级选择器
- 引导流程重构：选词书 → 选模式 → 学习量/复习方式
- 每日学习量扩展：预设 5/10/20/30/50/100 + 自定义输入框 (1-9999)
- **学习 & 复习统一 6 种模式**：闪卡 / 选择题 / 拼写 / 听写 / 配对 / 填空
- 学习 tab 解除 simple 限制，复习 tab 改用 DailyLearningMode，两 tab 均支持全部模式
- **页内一键切换模式**，无需返回引导向导，每种模式独立配色
- `reviewFocus` 属性区分标题（"学习模式" vs "复习模式"，"每日目标" vs "复习目标"）
- **切换词书按钮**：页面标题旁新增醒目词书切换卡片，可随时更换词书和模式
- 词书/模式记忆功能，刷新自动恢复

### 移动端适配 📱
