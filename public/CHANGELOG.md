# NativeThink 开发日志

## 2026-07-11

### 词书 & 引导流程
- 词汇深度切换后先选词书，再选模式（学习/复习/词库浏览/搭配学习）
- 学习模式：确定学习量和方式后进入，暂不显示调整控件
- 复习模式：支持 SM-2 已学复习 + 整本随机复习
- 所有 tab 统一使用 wizard 选定的词书，移除独立等级选择器
- 词库浏览和搭配学习只显示当前词书内容
- 添加词书记忆功能，刷新页面自动恢复上次选择的词书和模式

### 性能优化
- 词库按需动态导入，移动端只加载当前等级数据（28MB→按等级拆分）
- localStorage 缓存词库数据，二次访问秒开
- 导航悬停/触摸时预加载词汇数据
- 词列表分页加载（20条/页），减少 DOM 节点
- 加载骨架屏，数据未就绪时展示
- `dataVersion` 强制刷新 filteredWords，修复词书切换白屏

### 朗读功能
- 闪卡模式/学习模式：切换界面自动朗读，切换即停止上一次
- 听写模式自动朗读单词
- 搭配学习/短语库：选中短语自动朗读
- 拼写模式发音键移到输入框下方
- 修复朗读中断问题（ttsRef + lastSpokenKey 去重）

### 移动端适配
- 底部 Tab 导航栏
- 响应式布局（`col-span-12 lg:col-span-X`）
- 移动端间距优化（`px-4 py-4 pb-20`）
- 词库浏览筛选芯片紧凑布局
- Edge-TTS WebSocket + Google TTS 备用引擎
- 手机端 TTS 声音选择修复

### UI 改进
- 词列表页码点击跳转任意页
- 语块训练语块库分页 + 页码跳转
- 词库浏览添加筛选（词性/语域/情感/有搭配词/中文无对应）
- 每日一句收藏双向切换 + toast 反馈
- 历史记录追加保存所有换过的句子
