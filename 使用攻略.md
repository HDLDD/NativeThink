# NativeThink 使用攻略

> ⚠️ **温馨提示**：网站目前处于开发阶段，可能存在部分 bug。如遇到页面加载失败、朗读中断、数据未保存等问题，**请多试几次**（刷新页面或重新点击），大部分问题可以通过重试解决。感谢你的耐心！

---

## 快速开始

1. 打开 [nativethink.pages.dev](https://nativethink.pages.dev)
2. 首次使用建议**注册账号**（右上角登录按钮），可跨设备同步学习数据
3. 配置 **AI API Key**（设置 → AI 设置），解锁 AI 生成文章、翻译、造句等功能

---

## 各模块使用指南

### 📖 文章阅读

**阅读英文原著 / 刊物 / 演讲 / Wikipedia**

1. 选择上方标签：`书籍` / `刊物` / `AI 生成` / `演讲` / `维基百科`
2. 点击任意卡片 → 多页内容弹出**目录** → 选择章节开始阅读
3. 阅读器内：
   - **点击单词** → 弹出释义
   - **▶ 朗读** → 逐句播放当前页
   - **滚轮/按钮** → 翻页
   - **Aa** → 调整字号
   - **中/英/双语** → 切换显示模式
4. Wikipedia：搜索英文条目 → 点击阅读或 `+ 添加` 保存

**常见问题**：
- 维基百科加载失败 → 网络受限，多试几次或换关键词
- 朗读断断续续 → 点击暂停再重新播放
- 目录章节显示不完整 → 正常现象，点击任意章节即可开始阅读

---

### 📚 词汇深度

**学单词 + 复习 + 浏览 + 搭配**

1. 首次进入 → 选择词书 → 选择模式（学习/复习/浏览/搭配）
2. **学习**：6 种模式可选（闪卡/选择/拼写/听写/配对/填空），点击模式按钮切换
3. **复习**：SM-2 间隔记忆自动调度，紫色界面独立运营
4. **词库浏览**：左侧单词列表 + 右侧详情，支持筛选、搜索、自动朗读
5. **搭配学习**：左侧短语列表 + 右侧详情，点击 🧠 标记已记
6. 点击页面顶部 **切换词书** 可随时更换

**常见问题**：
- 切换 tab 后列表回到顶部 → 已修复，现在会记住位置
- 记忆功能偶尔失效 → 刷新页面后重试标记
- 词库数据加载慢 → 首次加载需要时间，后续访问秒开

---

### 🧩 语块训练

**短语学习 + 替换练习 + AI 对话**

1. **语块库**：左侧语块列表 + 右侧详情，支持记忆(🧠)和收藏(❤)
2. **自动朗读**：点击 `▶ 自动朗读` 依次朗读列表中所有语块
3. **AI 生成**：切换到 `AI 生成` 标签 → 选择分类/难度 → 生成专属语块
4. 点击语块进入详情 → 可查看用法、例句、AI 生成更多例句

**常见问题**：
- 页面加载报错 → 刷新页面重试
- 自动朗读中途停止 → 重新点击播放

---

### 🧠 母语思维训练

**AI 双语思维练习**

- 选择一个思维场景 → AI 同时给出英文和中文回答
- 适合练习"用英语思考"的能力

### 🎤 影子跟读

### ✍️ AI 写作练习

- 支持多种写作类型（议论文/书信/日记等）

---

### 💬 AI 对话练习

**与 AI 进行英文对话**

## 通用技巧

| 功能 | 操作 |
|------|------|
| 切换词书 | 词汇深度页面顶部绿色卡片 |
| 配置 AI | 右上角设置按钮 → AI 设置 |
| TTS 语音 | 右上角音量按钮 → 选择朗读声音 |
| 夜间模式 | 右上角太阳/月亮按钮 |
| 收藏 | 点击 ❤ 按钮 |
| 记忆标记 | 点击 🧠 按钮（搭配/语块/词库） |

---

## 已知问题 & 解决方案

| 问题 | 解决方案 |
|------|----------|
| 页面白屏/报错 | 刷新页面 (Ctrl+Shift+R) |
| Wikipedia 打不开 | 网络限制，多试几次或使用其他来源 |
| 朗读断续 | 暂停→重新播放，或刷新页面 |
| 语音不播放 | 检查浏览器是否允许音频，切换 TTS 引擎 |
| 数据未保存 | 刷新后重新操作，登录后可云同步 |
| 移动端显示异常 | 横屏或使用桌面版 |
| 词库加载慢 | 首次加载需等待，后续秒开 |
